# Radar Tracking System

## Overview

The Radar Tracking System is a C++ simulation that models how modern radar systems detect, track, and monitor multiple aerial targets.

This project is being developed as a professional software engineering portfolio project focused on embedded systems, robotics, and defense applications.

---

## Current Features

- Object-Oriented Design
- Target Class
- Radar Class
- Constructor Overloading
- Encapsulation
- Dynamic Target Storage using std::vector

---
